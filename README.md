# HackNation OpenAI Challenge

Monorepo that currently includes the backend API and data layer. Frontend and agent services can live alongside this when ready.

## Setup
```bash
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

## Dependencies
- Python 3.11+
- Packages listed in `requirements.txt`
