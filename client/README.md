# Client (Next.js)

This frontend is a Next.js + TypeScript MVP for the 45-second demo flow.

## Run

1. `pnpm install`
2. `pnpm run dev`
3. Open `http://localhost:3000`

## Demo flow

1. Click `Simulate Meeting Ended`
2. See the conflict in the Deconfliction Console
3. Click `Resolve`
4. Optionally run `What changed today?`

## Notes

- State is currently local mock data for speed and reliability.
- The code is organized for easy API integration to a real backend and knowledge graph.
